# 快速使用指南

## 系统访问

1. 启动开发服务器：
   ```bash
   npm run dev
   ```

2. 在浏览器中访问：http://localhost:5173 (或显示的端口)

3. 登录页面：
   - 输入任意用户名和密码即可登录（当前为模拟登录）
   - 例如：用户名 `admin`，密码 `123456`

## 功能使用说明

### 1. 首页（Dashboard）
- 查看系统数据概览
- 显示各模块统计数据
- 查看最近活动记录
- 快捷操作入口

### 2. 卡座管理
**卡座列表**
- 点击"添加卡座"按钮新增卡座
- 填写卡座编号、容纳人数、价格、描述
- 点击"编辑"修改卡座信息
- 点击"删除"移除卡座

**预订审核**
- 切换到"预订审核"标签页
- 查看待审核的预订列表
- 点击"通过"批准预订
- 点击"拒绝"拒绝预订

### 3. 组局管理
- 查看所有组局列表
- 点击"详情"查看组局完整信息
- 对待审核的组局进行审核（通过/拒绝）
- 删除不合适的组局

### 4. 商品管理
- 点击"添加商品"新增商品
- 填写商品名称、分类、价格、库存、图片URL、描述
- 点击"更新库存"快速调整库存数量
- 点击"编辑"修改商品信息
- 点击"删除"移除商品
- 库存低于20时会显示红色警告

### 5. 工作人员管理
- 点击"添加工作人员"新增员工
- 填写姓名、电话、职位、邮箱、入职日期
- 点击"编辑"修改员工信息（可修改状态为在职/离职）
- 点击"删除"移除员工记录

### 6. 兑换记录
- 查看所有兑换记录
- 点击"详情"查看兑换完整信息（包含收货地址）
- 对待审核的兑换进行审核（通过/拒绝）

## 注意事项

1. **数据持久化**
   - 当前所有数据都是模拟数据，存储在内存中
   - 刷新页面后数据会重置
   - 需要连接真实后端 API 才能持久化数据

2. **登录状态**
   - 登录信息存储在 localStorage 中
   - 关闭浏览器后登录状态会保持
   - 点击右上角头像 -> "退出登录"可退出系统

3. **表单验证**
   - 所有表单都有基本的验证规则
   - 必填项不能为空
   - 手机号和邮箱格式会进行验证

4. **响应式设计**
   - 系统支持不同屏幕尺寸
   - 在移动设备上侧边栏会自动折叠

## 后续开发

要连接真实后端 API，请按以下步骤操作：

1. 修改 `src/utils/request.js` 中的 `baseURL`：
   ```javascript
   baseURL: 'https://your-api-domain.com/api',
   ```

2. 修改 `src/pages/Login/index.jsx` 中的登录逻辑：
   ```javascript
   // 取消注释真实 API 调用
   const res = await login(values);
   setToken(res.data.token);
   setUser(res.data.user);
   ```

3. 在各个页面组件中，取消注释真实 API 调用，删除模拟数据

4. 根据后端返回的数据格式调整代码

## 技术支持

如有问题，请检查：
- 浏览器控制台是否有错误信息
- 网络请求是否正常
- Node.js 版本是否符合要求（建议 16+）
