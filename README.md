# 魔法王国后台管理系统

一个基于 React + Vite + Ant Design 的现代化后台管理系统。

## 功能模块

### 1. 登录页面

- 精美的登录界面设计
- 账号密码登录
- 表单验证

### 2. 卡座管理

- 卡座的增删改查
- 卡座状态管理（可用/已占用）
- 用户预订审核功能

### 3. 组局管理

- 组局列表展示
- 组局详情查看
- 组局审核（通过/拒绝）
- 组局删除

### 4. 商品管理

- 商品的增删改查
- 商品分类管理
- 库存管理
- 库存预警（低于20时显示警告）

### 5. 工作人员管理

- 员工信息的增删改查
- 职位管理
- 员工状态管理（在职/离职）

### 6. 兑换记录

- 兑换记录列表
- 兑换详情查看
- 兑换审核（通过/拒绝）

## 技术栈

- **React 18** - 前端框架
- **Vite** - 构建工具
- **Ant Design** - UI 组件库
- **React Router** - 路由管理
- **Axios** - HTTP 请求

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 默认登录信息

目前使用模拟登录，任意用户名和密码都可以登录。

在实际部署时，需要：

1. 修改 `src/pages/Login/index.jsx` 中的登录逻辑
2. 配置真实的后端 API 地址（在 `src/utils/request.js` 中修改 `baseURL`）
3. 实现各个模块的真实 API 调用

## 项目结构

```
src/
├── assets/          # 静态资源
├── components/      # 公共组件
│   ├── MainLayout.jsx      # 主布局
│   └── PrivateRoute.jsx    # 路由守卫
├── pages/           # 页面组件
│   ├── Login/       # 登录页
│   ├── Dashboard/   # 首页
│   ├── Booth/       # 卡座管理
│   ├── Activity/    # 组局管理
│   ├── Product/     # 商品管理
│   ├── Staff/       # 工作人员管理
│   └── Exchange/    # 兑换记录
├── services/        # API 服务
├── utils/           # 工具函数
│   ├── auth.js      # 认证相关
│   └── request.js   # HTTP 请求封装
├── App.jsx          # 应用入口
└── main.jsx         # 主文件
```

## 后续开发建议

1. **接入真实后端 API**
   - 修改 `src/utils/request.js` 中的 `baseURL`
   - 实现各个 service 文件中的 API 调用

2. **添加更多功能**
   - 数据导出功能
   - 图表统计
   - 权限管理
   - 操作日志

3. **优化用户体验**
   - 添加加载动画
   - 优化移动端适配
   - 添加更多交互反馈

4. **安全性增强**
   - 实现真实的 JWT 认证
   - 添加请求加密
   - 实现 RBAC 权限控制

## 注意事项

- 当前所有数据都是模拟数据，刷新页面后会重置
- 需要连接真实后端才能持久化数据
- 建议在生产环境中使用 HTTPS
- 记得修改默认的 API 地址和认证逻辑
